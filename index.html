Projects

CUDA Quantum to Qiskit Code Converter (Link)

This project bridges CUDA-Q and Qiskit by introducing an AST-based translation library, enabling seamless code conversion. It allows CUDA-Q users to access IBM’s quantum hardware without major rewrites, promoting interoperability, accelerating algorithm development, and fostering a more unified, efficient, and flexible quantum computing ecosystem across diverse platforms.

Psychoinformatics with R (Link)

To study human behavior, data was collected from 31 participants by asking them 25 questions related to their behavior. These questions were grouped into 5 categories Agreeableness, Conscientiousness, Extraversion, Neuroticism, and Openness with 5 questions in each category. A heatmap was generated in R to visualize the behavioral patterns.
